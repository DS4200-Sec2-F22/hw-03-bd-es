hw-03: Interactive Webpage, and Marks & Channels

Marks

  • Areas
    • This is a good choice because areas on a map represent different states in a clear and easy-to-understand fashion, as you can easily understand which areas represent which states.

Channels

  • Color (hue)
    • This is a good choice because differing hues are a perfect way to represent the separate identities that are voting parties.

Colormap(s)

  • Categorical
    • This is a good choice because the red and blue hues are commonly connotated with the conservative and democratic parties, so they can be easily understood.

Marks

  • Points
    • This is a good choice because each point is an individually batted ball from the 2015 season

Channels

  • Color (Hue)
    • This is a good choice because the different hues are easily differentiated from each other, and good/bad scoring values can be easily noticed.
  • Vertical and Horizontal Position
    • This is a good choice because the closer to the middle right the points are, the better the contact with the ball is.

Colormap(s)

  • Divergent
    • This is a good choice because the better scoring values can be easily distinguished from the others as dark orange, while the lower scoring values are darker purple. The median range is white, while the 'no data' representation is gray.
